Amazon Clone


I've developed a frontend clone of Amazon, focusing solely on replicating the user interface without backend functionality. Using HTML and CSS, I recreated the layout, navigation, and visual elements of the original site. While the clone lacks backend features such as user authentication and product management, it serves as a showcase of my front-end development skills.


Highlights:

  • Frontend Focus: Emphasized on crafting an intuitive user interface without backend functionalities.
  • HTML/CSS: Leveraged these technologies to mimic Amazon's layout and design.
  • Responsive Design: Ensured the clone's adaptability across various devices for optimal user experience.

Highlights:


  • Graphical Interface: The game features a user-friendly graphical interface designed with Windows Forms, providing a familiar environment for players to interact with.
  • Menu Options: The game includes various menu options, such as starting a new game, accessing information about the game, and exiting the application.
